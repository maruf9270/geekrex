Difference between Flex and Grid: The major difference between flex and grid is flex is one directional and grid is two directional. Flex can work with only row or column at a time but grid can work with both by using grid-template-column and grid-template-row

Bootstrap vs Tailwind: The main difference between Bootstrap and Tailwind is that Bootstrap pre-built components that provide a responsive UI (user interface) kit. On the other hand, Tailwind comes with predesigned widgets that facilitate web developers to design a website from scratch. Both Bootstrap and Tailwind are two of the best front-end CSS framework used worldwide. Tailwind has a huge components libary but bootstrap has less.

CSS box model:A rectangle box is wrapped around every HTML element. The box model is used to determine the height and width of the rectangular box. The CSS Box consists of Width and height (or in the absence of that, default values and the content inside), padding, borders, margin.
  • Content: Actual Content of the box where the text or image is placed.
  • Padding: Actual Content of the box where the text or image is placed.
  • Border:Area surrounding the padding.
  • Margin:Area surrounding the border.

Semantic Tag:Semantic tags clearly define the purpose of the HTML element. It also describes the type of content that the element should contain. For example, header, footer, article are considered semantic elements as they clearly describe their purpose and the type of content they should enclose.
